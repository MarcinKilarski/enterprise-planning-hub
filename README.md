# Enterprise Planning Hub

A professional archive of whitepaper summaries, tech evaluations, and executive reviews focused on AI-driven supply chain planning and enterprise value creation.

This repository uses [Jekyll](https://jekyllrb.com/) and is natively configured for deployment via [GitHub Pages](https://pages.github.com/).

├── _config.yml                     # Controls the site theme and settings

└── _posts/                         # Folder containing your 10 separate markdown articles

## Running Locally

To run the site locally, ensure you have Ruby and Bundler installed, then execute:

```bash
# Install dependencies
bundle install

# Run Jekyll server
bundle exec jekyll serve
```

Open `http://localhost:4000` in your browser.
